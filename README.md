# Bot TOA

Automatización de descarga de archivos desde TOA usando Selenium.

## Funcionalidades
- Login automático con SSO
- Descarga de archivos
- Procesamiento automático (watchdog)
- Renombrado y organización

## Estructura
- src/: código principal
- data/: archivos descargados y procesados
- logs/: registros

## Uso
```bash
pip install -r requirements.txt
python src/main.py
